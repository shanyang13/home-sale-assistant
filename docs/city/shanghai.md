## 城市定位

上海的定位（"上海2035"规划）：

- **经济**：国际经济、金融、贸易、航运中心
- **科技**：科技创新中心
- **文化**：国际文化大都市
- **目标**：卓越的全球城市，令人向往的创新之城、人文之城、生态之城

### 人口目标

- 至2035年：常住人口控制在2500万人左右
- 2015年常住人口：2415万人
- 人口增长空间有限，购房资格竞争加剧

### 名额到校/电脑派位

2022年起上海实行"名额到校"政策：
- 重点高中招生名额分配到各初中
- 弱校学生也有机会进重点高中
- 学区房逻辑发生变化

### 学区房特点

- **价格高**：优质学区房溢价明显
- **五年一户**：部分区域要求落户满5年
- **公民同招**：民办学校也需要摇号
- **政策变化**：名额到校政策影响学区价值

## 购房政策

### 限购政策

| 户籍 | 购房资格 |
|------|----------|
| 上海户籍单身 | 可购1套 |
| 上海户籍已婚 | 可购2套 |
| 非上海户籍 | 单身不可购，已婚可购1套（5年社保/个税） |
| 港澳台/外籍 | 可购1套（需境内无其他住房） |

### 首付比例

| 类型 | 首付要求 |
|------|----------|
| 首套房（普通住宅） | 35% |
| 首套房（非普通住宅） | 30% |
| 二套房（普通住宅） | 50% |
| 二套房（非普通住宅） | 70% |

### 普通住宅标准

| 条件 | 标准 |
|------|------|
| 容积率 | ≥1.0 |
| 单价 | ≤政府指导价（内环约10万/㎡） |
| 面积 | ≤140㎡ |

### 贷款政策

| 类型 | 利率 |
|------|------|
| 首套商业贷款 | LPR（4.3%） |
| 二套商业贷款 | LPR+60BP（4.9%） |
| 公积金贷款 | 首套3.1%，二套3.575% |

### 积分购房制度

热门新盘实行积分制：
- 基础分：户籍、房产、婚姻状况
- 社保分：缴纳年限
- 满分约80分
- 触发积分线的楼盘限售5年
