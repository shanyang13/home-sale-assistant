## 定价策略

### 定价参考因素

| 因素 | 说明 |
|------|------|
| 小区成交均价 | 参考近期同户型成交价 |
| 挂牌价vs成交价 | 一般成交价是挂牌价的 95%~98% |
| 楼层/朝向差价 | 优质楼层/朝向可上浮 5%~10% |
| 装修状况 | 精装/简装/毛坯有差异 |
| 急卖程度 | 急卖可适当降价 |

### 定价方法

#### 1. 市场比较法

```
参考价格 = 近期同户型成交价 × (1 + 楼层/朝向调整系数)
```

#### 2. 心理价位法

```
挂牌价 = 心理成交价 × 1.02~1.05（预留还价空间）
```

#### 3. 竞品参考法

列出 3~5 套同小区在售房源，分析各自优劣势，定出有竞争力的价格。

### 定价常见误区

| 误区 | 后果 |
|------|------|
| 价格虚高 | 无人问津，房源被标记为"不诚意卖" |
| 参考历史最高价 | 市场已下行，过时高价吓跑买家 |
| 只考虑自己买入价 | 市场价与买入价无关 |
| 不留还价空间 | 买家无法还价，直接跳过 |

### 定价建议

> **原则**：挂牌价应该是"有诚意谈"的价格，留 3%~5% 的还价空间。

## 挂牌技巧

### 多家挂牌策略

| 策略 | 适用情况 | 优缺点 |
|------|----------|--------|
| 全网挂牌 | 市场好时 | 曝光多，但中介可能互相压价 |
| 只挂一家独家 | 想省心 | 独家代理可能不力 |
| 精选2-3家 | 推荐 | 平衡曝光和中介积极性 |

### 挂牌渠道

| 渠道 | 特点 |
|------|------|
| 链家/贝壳 | 流量大，真实房源多 |
| 我爱我家 | 门店多，覆盖面广 |
| 安居客 | 分类信息平台，流量大 |
| 幸福里 | 字节系，流量增长快 |
| 闲鱼/朋友圈 | 私域流量，可能有惊喜 |

### 提升房源吸引力

1. **专业拍照**：请摄影师拍摄，视觉效果很重要
2. **VR全景**：部分平台支持VR看房，增加曝光
3. **户型图**：提供准确清晰的户型图
4. **亮点描述**：突出学区、地铁、装修等优势
5. **及时更新**：挂牌价、房屋状态有变化及时更新

## 筛选买家

### 判断真实买家

| 特征 | 可能是真实买家 | 可能是中介/不诚心 |
|------|----------------|-------------------|
| 来电 | 能说清需求，预约看房 | 问很多问题但不预约 |
| 看房 | 按时来，带家人看 | 放鸽子或临时取消 |
| 沟通 | 关心房屋具体情况 | 只问价格 |
| 资格 | 已了解购房资格 | 问限购政策等基础问题 |

### 了解买家情况

在谈判前，通过中介了解买家：

- 购房资格是否具备
- 付款方式（全款/贷款）
- 首付比例
- 购买紧迫程度
- 家庭结构（是否自住）

### 识别投资客

| 投资客特征 | 应对策略 |
|------------|----------|
| 问租金回报 | 强调居住价值 |
| 要求返租 | 谨慎，可能有税务问题 |
| 要求配合做低房价 | 拒绝，违法风险大 |
| 全款但压价狠 | 评估是否值得 |

## 谈判策略

### 把握谈判节奏

```
1. 买家出价 → 2. 评估是否达到底线 → 3. 适当坚持 → 4. 逐步让步 → 5. 成交
```

### 谈判要点

#### 1. 保持矜持

- 不要急于成交表现
- 让买家觉得你在考虑其他选项
- 可以适当表示"再考虑考虑"

#### 2. 组合谈判

除了价格，还可以谈：
- 交房时间（给你时间找新房）
- 家具家电（留下哪些）
- 户口迁出时间
- 提前还款解押时间

#### 3. 应对还价

**买家还价太低时**：
> "这个价格我确实卖不了，最近同小区的成交价你也应该看到了。如果诚心买，我们可以再谈谈。"

**买家犹豫时**：
> "我这边也有其他买家在谈，如果你真的想要，价格可以再商量。"

**多家出价时**：
> "现在有几个买家都在谈，我需要综合考虑，不只是看价格。"

### 价格谈判参考

| 情况 | 建议策略 |
|------|----------|
| 多人出价 | 选条件最好的，不一定选最高价 |
| 只有一家 | 评估买家资质和诚意 |
| 市场下行 | 尽快成交，落袋为安 |
| 市场上行 | 可以等等，但别贪心 |

## 签约注意事项

### 签约前确认

- [ ] 买家购房资格已核实
- [ ] 买家贷款预审批已通过（如需贷款）
- [ ] 买家付款方式确认（全款时间/贷款银行）
- [ ] 房屋无租约或租客已放弃优先购买权
- [ ] 水电物业费等已结清（可预留尾款结算）

### 合同必写条款

1. **成交价格**及付款方式、时间节点
2. **首付金额**及支付时间
3. **贷款金额**及审批时间
4. **过户时间**（具体日期）
5. **交房时间**及状态
6. **户口迁出时间**
7. **违约责任**（逾期付款、逾期过户等）
8. **家具清单**（哪些留下、哪些搬走）

### 卖房涉及费用

| 费用 | 承担方式 |
|------|----------|
| 增值税 | 通常卖家承担（满2年普通住宅免征） |
| 个人所得税 | 通常卖家承担（唯一住房满5年免征） |
| 中介费 | 买家或双方协商 |
| 赎楼费 | 如有贷款，买家承担或协商 |

### 收付款注意

1. **首付款**尽量走资金监管
2. **尾款**留 5~10 万交房当天结算
3. **避免**收取大量现金
4. **保留**所有转账记录

## 避免常见错误

### 定价错误

| 错误 | 后果 |
|------|------|
| 挂太高无人问津 | 房源热度下降，后期只能降价 |
| 挂太低卖亏了 | 后悔莫及 |
| 不留还价空间 | 吓跑诚心买家 |

### 选择买家错误

| 错误 | 后果 |
|------|------|
| 只看价格 | 付款能力差，过户出问题 |
| 不核实买家资质 | 买家没资格买，合同无效 |
| 被投资客套路 | 可能涉及做低房价等风险 |

### 合同错误

| 错误 | 后果 |
|------|------|
| 口头约定未写入合同 | 纠纷时无据可依 |
| 违约条款不对等 | 买家违约成本低 |
| 交房标准模糊 | 交房时扯皮 |

### 心态错误

| 错误 | 后果 |
|------|------|
| 总是觉得卖低了 | 反复违约，失去信誉 |
| 过于自信市场会涨 | 市场下行时损失更大 |
| 忽视买家感受 | 谈判破裂，错失成交 |
