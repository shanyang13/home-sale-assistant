## 谈判前的准备

### 信息收集

| 收集内容 | 获取渠道 |
|----------|----------|
| 小区近期成交价 | 贝壳找房、链家、安居客 |
| 挂牌历史价格 | 各大房产平台历史记录 |
| 房源基本信息 | 房产证、户型图、装修情况 |
| 房东售房原因 | 中介询问、侧面了解 |
| 市场行情 | 区域成交数据、涨跌趋势 |

### 制定谈判策略

1. **确定预算上限**：明确自己能接受的最高价格
2. **设定还价目标**：根据市场价制定合理还价幅度（一般 5%~10%）
3. **准备备选方案**：多看几套房子，避免被动
4. **了解房东心理**：售房原因急迫程度影响谈判力度

### 谈判时间选择

| 时机 | 优势 |
|------|------|
| 周中看房 | 购房者少，房东有时间详谈 |
| 月底/年底 | 中介冲业绩，促成意愿强 |
| 楼市淡季 | 买方市场，议价空间大 |
| 房东挂牌久 | 价格谈判空间更大 |

## 二手房谈判技巧

### 分析房东心理

| 房东情况 | 谈判策略 |
|----------|----------|
| 换房改善型 | 抓住时机，用"先买后卖"催促 |
| 资金周转型 | 可以压价，但需快速成交 |
| 投资获利型 | 看成本线，还有空间 |
| 移民/外迁型 | 通常急于出手 |
| 不急卖型 | 不宜强压价，保持关系 |

### 砍价技巧

#### 1. 出价策略

**低开高走**：不要一上来就出最高价，留出空间

**参考出价公式**：
```
合理价格 = 成交均价 × 0.95 ~ 1.0
极限价格 = 成交均价 × 0.9
```

#### 2. 找缺点压价

合理指出房源缺点：
- 楼层不好（顶层/底层）
- 采光通风一般
- 装修老旧需翻新
- 位置靠马路/铁路
- 房龄较老

> **注意**：不要人身攻击或过分贬低房源，保持专业和礼貌

#### 3. 组合谈判

除了价格，还可以谈：
- 家电家具留下
- 迁户时间
- 交房日期
- 过户费用分担

### 谈判流程

```
1. 中介约谈 → 2. 初次出价 → 3. 僵持拉锯 → 4. 让步成交 → 5. 签约
```

**要点**：
- 第一次见面不要急于成交
- 让对方先亮底牌
- 每次让步都要换取对方相应让步
- 保持冷静，不要情绪化

## 新房开盘选房策略

### 开盘前准备

1. **了解户型分布**：每个楼栋、每层的优劣势
2. **确定候选清单**：按优先级排列 3~5 个备选户型
3. **准备资金证明**：确保首付和贷款审批没问题
4. **关注开发商口碑**：口碑好的开发商更值得等待

### 选房优先级

| 优先级 | 选择因素 |
|----------|----------|
| 1 | 楼栋位置（不靠马路/垃圾房/变电站） |
| 2 | 楼层（避开设备层/腰线层/顶层） |
| 3 | 户型（方正、南北通、动静分区） |
| 4 | 朝向（南向优先，至少两个房间朝南） |

### 楼层选择建议

| 楼层 | 优点 | 缺点 |
|------|------|------|
| 低层（1-3层） | 方便老人小孩 | 采光差、潮湿、蚊虫 |
| 中层（4-12层） | 平衡采光和便利 | 价格较高 |
| 高层（13-24层） | 采光好、视野好 | 等电梯时间长 |
| 顶楼 | 视野最好、无楼上噪音 | 隔热防水问题 |

### 避开特殊楼层

- **设备层**：可能有噪音和震动
- **腰线层**：窗户可能变小，采光差
- **槽钢层**（超高层）：可能有渗水隐患
- **谐波层**：部分迷信说法的数字

## 谈判话术参考

### 初次见面

**中介常用开场**：
> "房东诚心卖，价格可以谈。"

**你的回应**：
> "我看过最近成交数据，这套房比同户型贵了XX万，您看能不能约房东坐下来聊聊？"

### 价格谈判

**错误示范**（暴露底牌）：
> "我最多能出到500万，再高我就不要了。"

**正确示范**（保留空间）：
> "根据我的了解和预算，出价480万，如果房东能在这个基础上谈谈，我们今天就可以定。"

### 僵持阶段

**房东不松口时**：
> "我理解房东的心情，但现在的市场你也知道，同小区最近成交价在那里摆着。如果今天能谈成，中介费这边我也可以提前准备好。"

**适当施压**：
> "我今天也看了另外一套，价格差不多，但那个位置更适合我。如果这边价格实在谈不拢，我可能就考虑那边了。"

### 促成签约

**中介施压时**：
> "价格我这边可以再让一步，但我想问一下，房东能不能把里面的空调和冰箱留下？这样大家都有诚意。"

## 常见陷阱与规避

### 价格陷阱

| 陷阱 | 表现 | 规避方法 |
|------|------|----------|
| 虚假挂牌价 | 低价吸引客户 | 多平台比较成交价 |
| 临时涨价 | 谈判中临时加价 | 口头协议后尽快签约 |
| 变相加价 | 捆绑车位/装修 | 明确各项费用明细 |

### 产权陷阱

| 陷阱 | 表现 | 规避方法 |
|------|------|----------|
| 共有产权 | 多人共有未经全部同意 | 签约前确认所有产权人 |
| 抵押查封 | 房子被抵押或查封 | 拉产调确认产权清晰 |
| 租赁备案 | 租客有优先购买权 | 确认是否有长租约 |

### 合同陷阱

| 陷阱 | 表现 | 规避方法 |
|------|------|----------|
| 补充协议 | 主合同外另有协议 | 所有约定写入合同 |
| 违约条款 | 违约责任不对等 | 仔细阅读违约条款 |
| 交房标准 | 交房条件模糊 | 明确交房时的状态 |

## 签约注意事项

### 签约前确认

- [ ] 房屋产权清晰，无抵押查封
- [ ] 所有产权人到场签字或公证委托
- [ ] 买家购房资格已核实（社保/户籍）
- [ ] 贷款预审批已通过
- [ ] 房价包含的所有物品清单确认（家具家电）

### 合同必写条款

1. **成交价格**及付款方式
2. **过户时间**（具体日期）
3. **交房时间**及状态（空屋/含家具）
4. **违约责任**（定金罚则、逾期责任）
5. **户口迁出时间**（如有）
6. **税费承担**（通常卖家增值税、个人所得税，卖家实收）

### 签约后注意

1. **定金支付**：一般不超过总价的 20%
2. **保留凭证**：所有付款记录留存
3. **跟进进度**：及时了解贷款审批、过户进度
4. **交房验收**：按照合同约定逐项验收
