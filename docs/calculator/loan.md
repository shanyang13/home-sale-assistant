## 贷款基本概念

### 贷款类型

| 类型 | 说明 | 利率 |
|------|------|------|
| 商业贷款 | 银行提供的贷款 | LPR+基点 |
| 公积金贷款 | 公积金中心提供的贷款 | 固定利率 |
| 组合贷款 | 商业+公积金混合 | 两种利率 |

### 还款方式

| 方式 | 特点 | 适用人群 |
|------|------|----------|
| 等额本息 | 每月还款额固定 | 收入稳定的人 |
| 等额本金 | 每月还本金固定，利息递减 | 前期还款能力强 |

### LPR（贷款市场报价利率）

LPR 是银行向最优质客户提供的贷款利率，由多家银行报价后计算得出。

**最新 LPR（2024年参考值）**：
- 1年期 LPR：约 3.45%
- 5年期以上 LPR：约 4.2%

> 具体数值请以央行最新公布为准

### 基点（BP）

1 基点 = 0.01%

例如：LPR+60BP = LPR + 0.6%

## 商业贷款计算

### 等额本息计算公式

```
月供 = [本金 × 月利率 × (1+月利率)^还款月数] ÷ [(1+月利率)^还款月数 - 1]
```

### 等额本金计算公式

```
月供 = 本金 ÷ 还款月数 + 剩余本金 × 月利率
```

### 计算示例

**条件**：
- 贷款金额：300万
- 贷款年限：30年（360个月）
- 利率：LPR 4.2%（即月利率 0.35%）

#### 等额本息

```
月供 = [3000000 × 0.0035 × (1+0.0035)^360] ÷ [(1+0.0035)^360 - 1]
     ≈ 14679 元
```

#### 等额本金（首月）

```
首月供 = 3000000 ÷ 360 + 3000000 × 0.0035
       = 8333 + 10500
       = 18833 元
```

等额本金每月递减约 `3000000 ÷ 360 × 0.0035 ≈ 29 元`

### 快速估算表

**贷款100万，30年，等额本息**

| 利率 | 月供 |
|------|------|
| 4.0% | 4774 元 |
| 4.3% | 4949 元 |
| 4.6% | 5126 元 |
| 4.9% | 5307 元 |

**贷款100万，30年，等额本金（首月）**

| 利率 | 首月供 |
|------|--------|
| 4.0% | 6111 元 |
| 4.3% | 6250 元 |
| 4.6% | 6389 元 |
| 4.9% | 6528 元 |

### 两种方式对比

| 对比项 | 等额本息 | 等额本金 |
|--------|----------|----------|
| 月供 | 固定 | 逐月递减 |
| 总利息 | 较多 | 较少 |
| 前期压力 | 小 | 大 |
| 适合人群 | 收入稳定 | 前期还款能力强 |

**举例**：贷款300万，30年，利率4.3%
- 等额本息：总利息约 228 万
- 等额本金：总利息约 194 万
- 差额约 34 万

## 公积金贷款计算

### 公积金贷款利率（参考值）

| 期限 | 利率 |
|------|------|
| 5年以下 | 2.6% |
| 5年以上 | 3.1% |

> 具体数值请以当地公积金中心公布为准

### 公积金贷款额度

各地计算方式不同，常见方式：

**方式一**：余额倍数
```
最高额度 = 公积金账户余额 × 倍数（各地不同，一般10~20倍）
```

**方式二**：缴存基数测算
```
最高额度 = 月缴存额 × 占比 × 12 × 贷款年限
```

**上海参考**：
- 首套房：最高100万（个人）或140万（家庭）
- 二套房：最高80万（个人）或110万（家庭）

**杭州参考**：
- 首套房：最高100万（职工）或150万（职工家庭）
- 二套房：最高80万

### 计算示例

**条件**：
- 公积金贷款：100万
- 贷款年限：30年（360个月）
- 利率：3.1%（月利率约 0.258%）

#### 等额本息

```
月供 = [1000000 × 0.00258 × (1+0.00258)^360] ÷ [(1+0.00258)^360 - 1]
     ≈ 4270 元
```

## 组合贷款计算

组合贷款 = 商业贷款部分 + 公积金贷款部分

### 计算示例

**条件**：
- 总贷款：200万
- 公积金贷款：60万（利率3.1%）
- 商业贷款：140万（利率4.3%）
- 贷款年限：30年

**计算**：
1. 公积金部分月供：约 2562 元
2. 商业部分月供：约 6929 元
3. **合计月供**：约 9491 元

## 贷款政策

### 首套房认定

| 城市 | 标准 |
|------|------|
| 大多数城市 | 名下无住房，且无贷款记录 |
| 上海 | 名下无住房 + 全国无公积金贷款记录 |
| 北京 | 名下无住房（认房不认贷） |

### 首付比例

| 类型 | 首付要求 |
|------|----------|
| 首套房 | 20%~30%（各地不同） |
| 二套房 | 30%~70% |
| 商铺/公寓 | 50%+ |

### 贷款年限

| 条件 | 最长年限 |
|------|----------|
| 住宅 | 30年 |
| 商用房 | 10年 |
| 房龄限制 | 50年 - 房龄 |

### 贷款年龄

| 条件 | 限制 |
|------|------|
| 男性 | 65岁 - 当前年龄 |
| 女性 | 60岁 - 当前年龄 |

## 提前还款

### 提前还款方式

| 方式 | 说明 |
|------|------|
| 全部还清 | 结清所有贷款 |
| 部分还清 | 归还部分本金 |

### 部分还清选择

| 选择 | 说明 |
|------|------|
| 缩短年限 | 月供不变，年限减少 |
| 减少月供 | 年限不变，月供减少 |

**建议**：如果前期还款能力强，选择"缩短年限"更省钱

### 违约金

| 银行 | 违约金标准 |
|------|------------|
| 大多数银行 | 1个月利息或无违约金 |
| 部分银行 | 3~6个月利息 |

> 提前还款前请咨询贷款银行

### 计算示例

**原贷款**：200万，30年，利率4.3%，已还3年

**提前还30万**：

| 选择 | 节省利息 |
|------|----------|
| 缩短年限 | 约 25 万 |
| 减少月供 | 约 10 万 |

## 常用计算公式速查

### 月供计算

```
月利率 = 年利率 ÷ 12

等额本息月供 = 本金 × 月利率 × (1+月利率)^期数 ÷ [(1+月利率)^期数 - 1]

等额本金月供(首月) = 本金 ÷ 期数 + 本金 × 月利率
```

### 可贷额度估算

```
可贷额度 ≈ 月缴存额 × 12 × 贷款年限 × 系数
```

### 收入要求

```
月收入 ≥ 月供 × 2
```

例如：月供5000元，收入需 ≥ 10000元
