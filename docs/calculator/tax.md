## 买家税费

### 契税

**计税依据**：房屋成交价（网签价）

**税率标准**（住宅）：

| 情况 | 税率 |
|------|------|
| 首套住房，90㎡以下 | 1% |
| 首套住房，90㎡以上 | 1.5% |
| 二套住房，90㎡以下 | 1% |
| 二套住房，90㎡以上 | 2% |
| 非住宅（商铺、公寓等） | 3% |

**计算公式**：
```
契税 = 成交价 × 税率
```

### 印花税

| 类型 | 税率 | 说明 |
|------|------|------|
| 住宅 | 0 | 目前免征 |
| 非住宅 | 成交价 × 0.05% | 减半征收 |

### 权属登记费

| 类型 | 费用 |
|------|------|
| 住宅 | 80~200元/件 |
| 非住宅 | 550元/件 |

### 评估费

| 条件 | 费率 |
|------|------|
| 住宅 | 评估价 × 0.1%（有上限，一般500~1000元） |
| 非住宅 | 评估价 × 0.3%~0.5% |

### 按揭服务费

| 类型 | 费率 |
|------|------|
| 商业贷款 | 贷款额 × 0.5%~1% |
| 公积金贷款 | 贷款额 × 0.3%~0.5% |

### 担保费

仅部分城市公积金贷款需要担保公司担保：
```
担保费 = 贷款额 × 0.3%~0.5%
```

## 卖家税费

### 增值税及附加

**计税依据**：房屋销售收入

**税率**：
```
增值税 = 销售收入 × 5%
附加税 = 增值税 × 12%
合计 = 销售收入 × 5.6%
```

**免税条件**：
- 个人将购买**2年以上**的住房对外销售，免征增值税
- 普通住宅满2年免征
- 非普通住宅按差额（卖出价 - 买入价）× 5%征收

### 个人所得税

**计税方式**：两种方式取低值

| 方式 | 计算公式 | 适用情况 |
|------|----------|----------|
| 核实征收 | （成交价 - 原价 - 合理费用）× 20% | 能提供原购入发票 |
| 核定征收 | 成交价 × 1%~2% | 无法提供原购入发票 |

**免税条件**：
- 个人唯一住房且满**5年**免征

### 土地增值税（商铺类）

仅对非住宅类房产（如商铺、写字楼、公寓）征收：

| 增值率 | 税率 | 速算扣除 |
|--------|------|----------|
| ≤50% | 30% | 0 |
| 50%~100% | 40% | 5% |
| 100%~200% | 50% | 15% |
| >200% | 60% | 35% |

### 中介费

| 类型 | 费率 |
|------|------|
| 链家 | 成交价 × 3%（单边） |
| 其他中介 | 成交价 × 1%~2% |

## 税费计算示例

### 示例一：买新房（300万，首套，90㎡以上）

| 费用项目 | 计算 | 金额 |
|----------|------|------|
| 契税 | 300万 × 1.5% | 4.5万 |
| 权属登记费 | - | 200元 |
| **合计** | - | **约4.52万** |

### 示例二：买二手房（500万，二手房，满2年）

**买家需付**：

| 费用项目 | 计算 | 金额 |
|----------|------|------|
| 契税 | 500万 × 2% | 10万 |
| 中介费 | 500万 × 2% | 10万 |
| 权属登记费 | - | 200元 |
| **买家合计** | - | **约20万** |

**卖家需付**：

| 费用项目 | 计算 | 金额 |
|----------|------|------|
| 增值税 | 免征（满2年） | 0 |
| 个人所得税 | 500万 × 1% | 5万 |
| 中介费 | 500万 × 2% | 10万 |
| **卖家合计** | - | **约15万** |

**注意**：通常卖家报价是"到手价"，所有税费由买家承担

### 示例三：卖非住宅（商铺，100万购入，200万卖出）

| 费用项目 | 计算 | 金额 |
|----------|------|------|
| 增值税 | 200万 × 5.6% | 11.2万 |
| 个人所得税 | (200-100)万 × 20% | 20万 |
| 土地增值税 | 增值50%，(200-100)×30% | 30万 |
| 中介费 | 200万 × 2% | 4万 |
| **合计** | - | **约65.2万** |

### 示例四：计算月供与税费总负担

**条件**：
- 房屋总价：500万
- 买家首套房，90㎡以上
- 商业贷款300万，30年，利率4.3%

| 项目 | 金额 |
|------|------|
| 首付（30%） | 150万 |
| 契税 | 7.5万 |
| 中介费 | 10万 |
| **前期支出** | **约167.5万** |
| 月供（等额本息） | 约14855元 |

## 各城市税费差异

### 主要城市税费政策对比

| 城市 | 个税征收方式 | 增值税免税条件 |
|------|--------------|----------------|
| 北京 | 1% 或差额20% | 满2年普通住宅免征 |
| 上海 | 1% 或差额20% | 满2年普通住宅免征 |
| 深圳 | 1.5% 或差额20% | 满2年免征 |
| 广州 | 1% 或差额20% | 满2年普通住宅免征 |
| 杭州 | 1% 或差额20% | 满2年免征 |

### 查询方式

- 北京：国家税务总局北京市税务局
- 上海：上海市税务局
- 深圳：深圳市税务局
- 各城市政务服务网

## 省钱技巧

### 1. 满五唯一

**定义**：房产证满5年，且是卖家唯一住房

**优惠**：免征增值税和个人所得税

**建议**：优先选择"满五唯一"的房源，可以省下大笔税费

### 2. 两年 vs 五年

| 条件 | 增值税 | 个税 |
|------|--------|------|
| 不满2年 | 需缴纳 | 需缴纳 |
| 满2年 | 免征 | 需缴纳 |
| 满5年唯一 | 免征 | 免征 |

**建议**：购买满2年的房子可以省增值税

### 3. 房价与税费的平衡

有时高房价但满五唯一的房子，总成本反而更低：

```
总价500万，满2年：税费约 17.5万（买家承担）
总价480万，不满2年：税费约 27万（买家承担）

实际差额：480万+27万 = 507万 > 500万+17.5万 = 517.5万
```

### 4. 合理利用首套资格

| 首套认定 | 契税 |
|----------|------|
| 90㎡以下 | 1% |
| 90㎡以上 | 1.5% |
| 二套 | 3% |

**建议**：首套房尽量买大一点，避免浪费首套资格

### 5. 卖家实收 vs 买家包税

**卖家实收**：卖家到手价，所有税费由买家承担

**买家包税**：成交价是纯到手价，税费另算

**建议**：
- 如果市场是买方市场，争取"买家包税"
- 无论哪种方式，都要算清楚总成本
