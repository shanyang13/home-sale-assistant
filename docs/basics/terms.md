## 基础概念

### 容积率

**定义**：容积率是指一个小区的地上总建筑面积与用地面积的比率。

**计算公式**：
```
容积率 = 总建筑面积 ÷ 用地面积
```

**影响**：容积率越高，说明小区内房子盖得越多，居住密度越高，舒适度可能降低。

**参考数值**（住宅）：

| 容积率 | 居住体验 |
|--------|----------|
| 0.5~1.0 | 低密度别墅区 |
| 1.0~2.0 | 舒适住宅区 |
| 2.0~3.0 | 普通高层住宅 |
| 3.0~4.0 | 密集高层住宅 |
| > 4.0 | 超高层/公寓 |

---

### 绿地率

**定义**：绿地率是指小区用地范围内各类绿地的总和与用地面积的比率。

**计算公式**：
```
绿地率 = 绿地面积 ÷ 用地面积 × 100%
```

**参考标准**：
- 新建住宅小区绿地率应 ≥ 30%
- 绿化环境较好的小区绿地率可达 40%~50%

---

### 公摊面积

**定义**：公摊面积是指由整栋楼的产权人共同分摊的整栋楼公用部分的建筑面积。

**包括范围**：
- 电梯井、管道井、楼梯间、垃圾道、变电室、设备间、公共门厅、过道、地下室、值班警卫室等
- 套与公共建筑之间的分隔墙，以及外墙（包括山墙）水平投影面积一半的建筑面积

**不包括范围**：
- 车位面积
- 人防工程面积
- 架空面积

**公摊率参考**：

| 楼层类型 | 公摊率 |
|----------|--------|
| 多层（6层以下） | 10%~15% |
| 小高层（7-11层） | 15%~20% |
| 高层（12层以上） | 20%~30% |
| 超高层 | 25%~35% |

---

### 得房率

**定义**：得房率是指套内建筑面积与销售面积的比率。

**计算公式**：
```
得房率 =（套内建筑面积 ÷ 销售面积）× 100%
       = [（销售面积 - 公摊面积） ÷ 销售面积] × 100%
```

**参考数值**：

| 类型 | 得房率 |
|------|--------|
| 多层住宅 | 85%~90% |
| 小高层住宅 | 80%~85% |
| 高层住宅 | 70%~80% |
| 超高层 | 65%~75% |

> **提示**：得房率不是越高越好，需要综合考虑楼层高度、配套设施等因素。

---

### 楼层净高

**定义**：楼层净高是指楼面或地面至上部楼板底面之间的最小垂直距离。

**国家标准**：
- 住宅卧室、起居室（厅）的室内净高不应低于 2.40m
- 厨房、卫生间的室内净高不应低于 2.20m
- 局部净高不应低于 2.10m（斜顶等特殊情况）

**为什么默认层高是3米左右？**
- 结构层厚度约 0.2m
- 装修地面约 0.05m
- 净高约 2.75m，接近 2.80m 国标要求
- 考虑吊顶、管线等空间

## 用地性质

根据《城市用地分类与规划建设用地标准》，城市建设用地分为以下类别：

### R 类 - 居住用地

| 类别 | 说明 |
|------|------|
| R1 一类居住用地 | 设施齐全、环境良好，低层住宅为主 |
| R2 二类居住用地 | 设施较齐全、环境较好，多/中/高层住宅 |
| R3 三类居住用地 | 设施欠缺、简陋住区（危房、棚户区等） |

### B 类 - 商业服务业设施用地

| 类别 | 说明 |
|------|------|
| B1 商业设施用地 | 零售、餐饮、旅馆等 |
| B2 商务设施用地 | 金融保险、艺术传媒、技术服务等 |
| B3 娱乐康体设施用地 | 剧院、电影院、高尔夫等 |

### M 类 - 工业用地

| 类别 | 说明 |
|------|------|
| M1 一类工业用地 | 对居住和公共环境基本无干扰 |
| M2 二类工业用地 | 有一定干扰和污染 |
| M3 三类工业用地 | 严重干扰和污染 |

### A 类 - 公共管理与公共服务设施用地

| 类别 | 说明 |
|------|------|
| A1 行政办公用地 | 党政机关、事业单位等 |
| A3 教育科研用地 | 高校、中小学、科研单位等 |
| A4 体育用地 | 体育场馆、训练基地等 |
| A5 医疗卫生用地 | 医院、卫生防疫等 |

### U 类 - 公用设施用地

供水、供电、供燃气、供热、排水、邮政、广播电视等设施用地。

### G 类 - 绿地与广场用地

| 类别 | 说明 |
|------|------|
| G1 公园绿地 | 向公众开放，游憩为主 |
| G2 防护绿地 | 卫生、隔离、安全防护 |
| G3 广场用地 | 公共活动场地 |

## 房屋类型

### 按高度分类

| 类型 | 楼层数 |
|------|--------|
| 低层 | 1~3层 |
| 多层 | 4~6层 |
| 小高层 | 7~11层 |
| 高层 | 12~30层 |
| 超高层 | 30层以上 |

### 按结构分类

| 类型 | 说明 |
|------|------|
| 砖混结构 | 造价低、保温好，但抗震性差 |
| 框架结构 | 抗震性好，空间分隔灵活 |
| 框架剪力墙结构 | 高层住宅常用，抗震性强 |
| 钢结构 | 强度高、自重轻，多用于超高层 |

### 按产权分类

| 类型 | 说明 |
|------|------|
| 大产权（商品住房） | 70年产权，正常上市交易 |
| 小产权 | 农村集体土地上建设，无法正常交易 |
| 商业办公 | 40年产权，水电费较高 |

## 户型相关

### 东边套、西边套、中间套

| 类型 | 特点 |
|------|------|
| 东边套 | 上午采光，三面采光，通风好，无西晒；雨季可能潮湿 |
| 西边套 | 下午采光，冬季温暖，价格可能较低；有西晒问题 |
| 中间套 | 南北通透户型通风好，价格相对便宜；只有两面采光 |

### 户型朝向

| 朝向 | 特点 |
|------|------|
| 正南 | 采光最佳，冬季温暖 |
| 东南 | 采光好，通风佳 |
| 正东 | 上午采光，夏季凉快 |
| 西南 | 下午采光，冬季温暖，但可能有西晒 |
| 正西 | 西晒严重，夏季较热 |
| 西北 | 采光差，冬季寒冷 |

### 户型方正

指户型的长宽比例接近1:1，进深与开间之比合理（一般不超过1.5:1）。

**优点**：
- 空间利用率高
- 家具摆放方便
- 视野开阔

## 交易相关

### 认筹

开发商在开盘前收取购房者"认筹金"，获得优先购房权。认筹金可退，但可能被挪用。

### 摇号

当购房人数超过房源数时，采用随机抽签方式确定购房资格。

### 网签

房地产开发商与购房者签订商品房预售合同后，到房地产相关部门进行备案，并公布在网上。

### 面签

购房者到银行签订贷款合同，确认贷款金额、利率、还款方式等。

### 过户

房屋产权从卖方转移到买方的登记过程。过户完成后，购房者正式获得房屋产权。

### 红本与绿本

| 类型 | 说明 |
|------|------|
| 红本 | 商品房房产证，可上市交易 |
| 绿本 | 政策性住房房产证，如经济适用房，需满一定年限才能交易 |
